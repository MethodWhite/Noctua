# Noctua

Framework de **Reverse Engineering** en Python.

## Características

- **7 loaders**: ELF, PE, Mach-O, DEX, WASM, WebP, Generic
- **15 módulos**: BranchTiming, Dataflow, Spectral, Crypto, etc.
- **Pipeline modular**: Stages independientes
- **Clean Architecture**: Domain/Application/Infrastructure/Interface
